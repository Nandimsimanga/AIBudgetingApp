# AI Budget Tracker

Mobile-first web app prototype for expense tracking, savings goals, financial reports, and AI-powered budgeting insights.

**Live demo:** https://nandimsimanga.github.io/AIBudgetingApp/

## Local development

```bash
npm install
npm run dev
```

## Deploy (GitHub Pages)

Pushes to `main` deploy automatically via [GitHub Actions](.github/workflows/deploy.yml).

1. Push this repo to [github.com/Nandimsimanga/AIBudgetingApp](https://github.com/Nandimsimanga/AIBudgetingApp)
2. In the repo go to **Settings → Pages**
3. Under **Build and deployment**, set **Source** to **GitHub Actions**
4. After the workflow completes, the site is live at the URL above

## Stack

- React + Vite
- Recharts
- Lucide icons
- localStorage persistence
